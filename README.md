# Gesture-Control
Interactive 3D Universe controlled by real-time hand gestures. Built with Three.js and MediaPipe, this web app allows users to morph particle systems into hearts, galaxies, and planets using only their camera. Includes pinch-to-color, rotation, and supernova effects.
